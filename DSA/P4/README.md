# Project: Escalator - Stack Implementation

## Overview
This is a C++ lab project focused on implementing a Stack using an array. The project fulfills all the specified requirements, including object-oriented programming concepts such as Class & Object, Polymorphism, Inheritance, and Encapsulation.

## Features & Implementation
- **Class & Object**: Defined a `Stack` class and created its object to manage stack operations.
- **Encapsulation**: Used `private` access modifiers for the array and top index variables to prevent direct access, exposing only necessary operations via `public` methods.
- **Inheritance**: Inherited the `Stack` class from an abstract base class `IStack`.
- **Polymorphism**: Used virtual functions in the `IStack` interface which are overridden by the `Stack` class, and demonstrated by calling methods through a base class pointer.

## Expected Functionality
1. **Push an element**: Prompts the user for a value and adds it to the stack if it is not full.
2. **Pop an element**: Removes the top element from the stack if it is not empty.
3. **Display the top element**: Displays the top element without removing it.
4. **Check if the stack is empty**: Outputs whether the stack is empty.
5. **Check if the stack is full**: Outputs whether the stack is full.
